WHAT IT DOES

01
Instant URL Cleaning
Every URL is cleaned before it loads. Tracking parameters stripped in real time — no redirect, no reload, no visible interruption.
02
Per-Domain Whitelist New
Exclude any domain from cleaning in one click. Stop disabling the entire extension because a single site misbehaves.
03
Custom Rules New
Write rules that override built-in or remote sources. Disable specific rules without creating counter-rules — then restore them anytime.
04
Multiple Rule Sources New
Load and merge several remote rule sets simultaneously. Overload mode merges built-in and remote rules before applying your custom rules on top.
05
History API Protection
Catches URLs that bypass network-level rules entirely by being pushed via the browser's History API.
06
ETag & Ping Blocking
Blocks hyperlink auditing pings and ETag-based tracking across sessions — attack vectors most extensions ignore.
07
Search Engine Fix
Strips tracking wrappers injected into Google and Yandex search results before you ever click a link.

0

  • No telemetry
  • No analytics
  • No usage tracking
  • 100% local processing

All processing happens in your browser.

Linkumori never phones home. URL cleaning runs entirely inside your browser's extension runtime. Nothing you visit is logged, reported, or analyzed — not even anonymously.

Remote rule sources are disabled by default. They're an opt-in feature for advanced users who run their own infrastructure. When enabled, you point Linkumori at a server you control — no third party has visibility.
